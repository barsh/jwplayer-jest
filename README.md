# jwplayer-jest

An attempt to load jwplayer with jsdom for testing with jest.

## Setup

```sh
yarn install
```

## Running Tests

```sh
# all tests
yarn test

# watch
yarn test --watch

```

## Debugging

You must be running Node v9.2.0 or higher.

1. Open repo folder in Visual Studio Code

1. Set a breakpoint or insert `debugger` command in `./tests.ts`

1. Press <kbd>F5<kbd>
